# Tea Social - Backend API

Серверная часть социальной сети **TeaSocial**, построенная на стеке Java + Spring Boot. Бэкенд спроектирован по классической многослойной архитектуре (Layered Architecture) с полной изоляцией доменных сущностей (Entities) от слоя передачи данных (DTO).

---

## 🛠️ Технологический стек & Архитектура

*   **Core:** Java 17, Spring Boot 3.x (Spring Web, Spring Security)
*   **Data Layer:** Spring Data JPA, Hibernate ORM
*   **Database:** PostgreSQL
*   **Mapping:** MapStruct 
*   **Security:** Spring Security + JWT Authentication
*   **Build Tool:** Maven
